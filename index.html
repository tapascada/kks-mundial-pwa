Sincronizar Google Drive

Ingresa el ID del archivo de Google Drive generado por la app Kikes Mundial (o el enlace para compartir completo).

¿Cómo obtener el enlace?

  1. Exporta el archivo de posiciones desde la app WPF.
  2. En Drive web, haz clic derecho sobre el archivo -> Compartir -> Obtener enlace.
  3. Cambia el acceso a "Cualquier persona con el enlace puede ver".
  4. Copia el enlace y pégalo aquí. Extraeremos el ID automáticamente.
